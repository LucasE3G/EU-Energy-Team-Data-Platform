# E3G Logo Setup

To use the E3G logo in the application:

1. Copy your E3G logo image file to the `assets` folder
2. Name it `e3g-logo.png`
3. The application will automatically display it in the sidebar and top bar

If the logo file is not found, the application will display "E3G" as text instead.

## Logo Colors Used in Design

The application uses E3G brand colors:
- **Dark Indigo Blue**: #2B2E51 (Primary)
- **Light Periwinkle Blue**: #87ADD3 (Secondary)
- **Golden Yellow/Ochre**: #BDB068 (Accent)
- **Black**: #000000 (Sidebar background)
